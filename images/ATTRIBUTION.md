# Botanical artwork

`botanical-bellflower.webp` is derived from *A guide to the wild flowers (Plate XLII)* (1899), illustrated by Marian Ellis Ryan Rowan and sourced from the Biodiversity Heritage Library via Wikimedia Commons.

The source scan is marked as public domain: https://commons.wikimedia.org/wiki/File:A_guide_to_the_wild_flowers_(Plate_XLII)_BHL23798277.jpg

`botanical-rose-redoute.webp` is derived from Pierre-Joseph Redouté's *Rosa gallica purpuro-violacea magna* (1824). The source work is marked as public domain: https://commons.wikimedia.org/wiki/File:Redoute_-_Rosa_gallica_purpuro-violacea_magna.jpg

`botanical-peony-line.webp` is derived from the Pearson Scott Foresman line drawing *Peony (PSF)*. The source is released into the public domain: https://commons.wikimedia.org/wiki/File:Peony_(PSF).png

The local web assets are cropped, optimized, and—in the case of the peony—recolored to the VETKA olive palette for decorative use.
