# Букет дня из Telegram

Сайт уже умеет получать отмеченный пост из канала `@vetka_2024` и показывать его в первой карточке главного экрана.

## Что нужно один раз

1. Добавить существующего бота заявок администратором канала `@vetka_2024` с правом видеть публикации.
2. В окружении сайта оставить текущие `BOT_TOKEN`, `CHAT_ID`, `KV_REST_API_URL`, `KV_REST_API_TOKEN` и добавить:
   - `TELEGRAM_CHANNEL_ID=@vetka_2024`
   - `TELEGRAM_WEBHOOK_SECRET` — случайная строка длиной 32–64 символа.
3. После публикации сайта выполнить настройку webhook с теми же значениями `BOT_TOKEN`, `TELEGRAM_WEBHOOK_SECRET` и `SITE_URL`:

   `npm run telegram:webhook`

Перед подключением стоит проверить `getWebhookInfo`: у Telegram-бота может быть только один webhook. Код заявок сайта webhook не использует, поэтому внутри этого проекта конфликта нет.

## Как публиковать

В канале нужен пост с фотографией и отметкой `#букетдня` или фразой «Букет дня». Пример:

```text
#букетдня
Нежный сезонный букет
Воздушная форма с каллами и полевыми акцентами.
8 500 ₽
```

Сайт возьмёт фото, название, описание, цену и ссылку на исходный пост. Редактирование поста также обновит карточку.
